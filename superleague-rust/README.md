# SuperLeague in Rust

## Requirements

- Rust Toolchain (via [Rustup.rs](https://rustup.rs/))
- [Serde JSON](https://docs.rs/serde_json/latest/serde_json/)

## Run It

    cargo run -- data/super-league.json
    cargo run -- data/challenge-league.json
    cargo run -- data/sorting.json
